# 专网环境尚未完全落实事项

以下事项按是否阻塞正式专网投产进行分层。

## 阻塞正式投产

1. 管理面认证与授权仍未形成完整闭环，当前更适合受控内网。
2. 高危运维动作（重启、保护恢复）缺少强身份绑定与审批。
3. 敏感配置加密存储、密钥轮换、审计签名尚未完成。
4. Smoke 脚本尚未在专网 Windows 环境形成稳定通过基线。

## 非阻塞但建议投产前完成

1. 专网时钟同步与证书/签名时间偏差治理。
2. 双网卡/专用管理网卡绑定策略与冲突检测。
3. 管理面 CIDR 白名单与跳板机访问基线。
4. 断网、弱网、端口漂移、DNS 不可用场景下的恢复演练。
5. 专网下对端平台互通矩阵与验收记录。

## 已具备基础但仍需演进

1. 链路状态机、就绪探针、/metrics、保护状态 UI。
2. 三层保护（全局/映射/来源 IP）及热点对象统计。
3. 熔断显式状态与手工恢复入口。
4. 访问日志、审计、诊断导出与专网运维说明。
