# P2 保护与熔断落地

- 入口保护维持全局 RPS / Burst / MaxConcurrent。
- 运行态新增热点对象统计：限流命中 Top、并发拒绝 Top、放行最多 Top。
- 告警与保护页直接展示这些运行态对象，便于快速定位是单个 mapping、单个来源还是全局流量触发保护。
- 建议调优顺序：先确认热点对象，再调 Burst / MaxConcurrent，最后再看是否需要扩容或拆分流量。


## P2 完整落地项

- 分层保护（全局 / 按映射 / 按来源 IP）同时生效。
- 并发保护与速率限制分开统计，并暴露到 `/api/protection/state` 与 `/metrics`。
- 管理台“告警与保护”页面展示每一层的实时阈值、活跃数、命中数和热点对象。
- 运行态热点对象可直接用于排查单映射或单来源 IP 的突发流量。
